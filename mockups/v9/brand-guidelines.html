Logo Dos & Don'ts

  • Always maintain adequate clear space around the logo
  • Use approved color variations only
  • Scale proportionally; never stretch or distort
  • Never rotate the logo
  • Never add effects like drop shadows or outlines

Voice & Tone

Our writing is conversational, confident, and human-first. We avoid jargon and always put the user first.

✓ Do
  • Be conversational and approachable
  • Use active voice
  • Focus on user benefit
  • Be honest and transparent
  • Use simple, clear language
✗ Don't
  • Use corporate jargon
  • Be overly formal or stuffy
  • Make unsupported claims
  • Use passive voice
  • Assume user knowledge

Usage Patterns

✓ Effective Usage
  • Use neon yellow to highlight key information
  • Maintain high contrast for accessibility
  • Keep layouts spacious and uncluttered
  • Use consistent border radius (8px minimum)
  • Apply animations sparingly and purposefully
✗ Common Mistakes
  • Using neon colors for body text
  • Overcrowding the layout
  • Inconsistent rounded corners
  • Low contrast text on dark backgrounds
  • Overusing animations
