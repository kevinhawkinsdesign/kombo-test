**Method note:** all five help centers block direct automated fetching (HTTP 403), so briefs were assembled from search-indexed article content plus corroborating third-party sources. Cited article URLs are real help-center pages. Spot-check live pages before quoting externally.

**Cross-cutting findings**

1. **The conversation layer is open** — none of the five document live in-call coaching. Lusha Conversations is post-call recording; Lemlist's Claap is a one-way bolt-on.
2. **Credit fine print is universal** except Lemlist, the only success-only charger (failed lookups free).
3. **"Real-time CRM sync" doesn't exist in this field** — Amplemarket pushes HubSpot 3-hourly, Clay imports daily, Lemlist lags 2–10 min with non-retroactive mappings.
4. **MCP servers are becoming table stakes** — Lusha, Clay, and Enginy all ship one.
5. **Enginy is an EU positioning threat, not a feature threat** — demo-gated €799/mo, no call layer, no SOC 2.
