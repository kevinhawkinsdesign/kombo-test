**Cross-cutting findings**

1. **The conversation layer is open** — none of the seven document live in-call coaching (Apollo's Conversations is post-call; LinkedIn's Sales Assistant drafts, it doesn't coach). Lusha Conversations is post-call recording; Lemlist's Claap is a one-way bolt-on.
2. **Credit fine print is universal** except Lemlist, the only success-only charger (failed lookups free).
3. **"Real-time CRM sync" doesn't exist in this field** — Amplemarket pushes HubSpot 3-hourly, Clay imports daily, Lemlist lags 2–10 min with non-retroactive mappings.
4. **MCP servers are becoming table stakes** — Lusha, Clay, and Enginy all ship one.
5. **Enginy is an EU positioning threat, not a feature threat** — demo-gated €799/mo, no call layer, no SOC 2.
6. **Apollo wins on breadth-per-dollar but is capped by its own flywheel** — ~65% measured accuracy, expiring/double-billing credits, post-call-only coaching, and a LinkedIn-banned extension.
7. **LinkedIn Sales Navigator is the substrate, not a displaceable rival** — SNAP API frozen (Aug 2025), suspension-first automation enforcement (Q1 2026); any layer on top must be strictly assistive.
